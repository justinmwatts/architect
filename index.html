Architect

Responsive mobile first CSS framework with global CSS settings, default HTML element styling and an advanced grid system. Follows the ITCSS (Inverted Triangle CSS) philosophy and BEM methodology. Built with SASS.

Grid system

The grid column amount can easily be changed to any desired amount. The grid is set to 12 columns by default. In _settings.global.scss, use $grid-cols to update the column amount. Column gutters are set to 10px by default. In _settings.global.scss, use $grid-col-gutter-width to update the gutter width.
